## 🔄 重新启动

运行以下命令：
```bash
./check-and-run.sh
```

或使用 nvm 环境：
```bash
./run-with-nvm.sh npm run dev
```

## 📝 注意事项

- Node.js 通过 nvm 安装，版本: v24.12.0
- 如果关闭终端后需要重新启动，请先加载 nvm 环境：
  ```bash
  source ~/.nvm/nvm.sh
  ```
- 或者直接运行 `./check-and-run.sh`，脚本会自动加载 nvm
